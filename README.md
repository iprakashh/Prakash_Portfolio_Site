# Prakash Portfolio Website

Personal portfolio website showcasing my experience as a QA Engineer and Automation Tester.  
The site highlights my professional experience, projects, certifications, and allows visitors to contact me directly.

🌐 *Live Website:*  
https://prakash-portfolio-site.onrender.com

---

## About the Project

This portfolio website was built to present my QA automation skills, professional background, and projects in a simple and modern web interface.

The website includes sections such as:

- About Me
- Skills
- Professional Experience
- Projects
- Certifications
- Contact Form

Visitors can explore my work and send messages through the contact form.
